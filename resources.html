Config files

  • surf.cfg

    Put the .cfg into the folder: C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive\csgo\cfg. In-game in a surf map (except for the fall games map) run the config with "exec surf" in the console.

  • kz.cfg

    Put the .cfg into the folder: C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive\csgo\cfg. In-game in a kz map (except for the csgo_hub map) run the config with "exec kz" in the console. Setting a chekpoint is bound to key "8", and teleporting to back to the checkpoint is bound to "9" by default and can be changed in the .cfg .
